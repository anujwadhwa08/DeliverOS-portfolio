What the Driver App Does
📋 Orders & Dispatch
  • Go online / offline with one toggle
  • See available orders with total, type & distance
  • Accept or decline — auto-reassigned on decline
  • No race conditions: first confirmed write wins
  • Real-time status updates every 15 seconds
🔢 Secure Handover
  • Unique 6-character code per order
  • Code only shown at DriverAtRestaurant state
  • Restaurant enters code on their end to confirm
  • Code match required to advance order — no skipping
  • Admin can override handover if code is lost
💰 Earnings & Payouts
  • Available balance: withdrawable minus $20 reserve
  • Held balance: 15% held 7 days for dispute coverage
  • Lifetime earnings & delivery count tracked
  • Weekly / monthly / yearly breakdown
  • Stripe Connect payout — request anytime
