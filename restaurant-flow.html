The restaurant app covers the complete journey: guided AI-powered onboarding, instant menu setup (from URL, photo, or spreadsheet), live order management with a real-time countdown, and secure driver handover via a 6-character code. Available on iOS, Android, web, and Telegram.

Menu Setup During Onboarding — URL Import
Zero-friction menu setup. During onboarding, restaurants paste any menu URL — the AI scrapes, structures, and categorises every item automatically. Also supports photo upload and Excel import. A 100-item menu goes live in under 5 minutes with no manual data entry.
Order Management — 6 Screens from the Running App
1
Live Order Board
All incoming orders in real time — order contents, customer name, delivery address. New orders pulse to draw attention.
2
Confirm + Set Prep Time
Tap to confirm. Enter prep time in minutes — the countdown starts immediately and the customer sees the ETA update.
3
Prep Countdown + Mark Ready
Live countdown timer visible to both restaurant and customer. Restaurant taps ‘Ready’ when food is packed — triggers driver dispatch.
4
Driver at Door
When the driver arrives to collect, restaurant sees a ‘Verify Driver’ prompt. Tapping opens the code entry modal.
5
Handover Code Entry
Restaurant enters the 6-character code shown on the driver’s phone. Code is single-use and expires after entry — confirms physical handover.
6
Completed Orders
Completed order history with timestamps, items, and payout status. Shows when the Stripe transfer was initiated.
Menu Management — Import, Browse & Edit
1
Menu Browser
Full menu with categories, items, prices, and availability toggles. Restaurant can mark individual items as unavailable without removing them.
2
Import from Website URL
Paste any restaurant website URL — the AI scrapes the menu, infers categories, and creates all items. Review and confirm before publishing.
3
Web Portal — Excel Menu Import
The web portal’s Excel import tool lets restaurants download a pre-formatted template, fill it in, and upload — all items and categories are created in one shot. No manual entry.
Restaurant Web Portal — Orders & Analytics
Full browser experience. The web portal gives restaurant owners a full view of live orders, daily revenue, completed order history, and menu status — all in a browser without the mobile app. The same order state machine drives both the app and portal — an order confirmed on Telegram appears in the web portal instantly.
Telegram Channel — Same Experience, No App Required

Restaurants can manage their entire workflow via the Telegram bot — receive orders, confirm with prep time, mark ready, and verify driver handover — all through chat. The Telegram conversation and mobile app share the same order state — confirming an order via Telegram updates the customer’s tracking screen in real time.

Feature Highlights
Order Management
Live order board with real-time updates. Confirm with custom prep time. Countdown visible to customer. One-tap confirm or reject — decline triggers automatic cancellation and refund.
Secure Handover
6-character single-use code on driver’s phone. Restaurant staff enters it to confirm physical handover. Prevents false delivery claims.
Menu & Onboarding
AI imports menu from URL, photo, or spreadsheet. Categories created automatically. Availability toggles per item. Opening hours and pausing built in.
