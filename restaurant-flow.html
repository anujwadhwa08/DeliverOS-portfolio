What the Restaurant App Does
📋 Order Management
  • New orders surface instantly with red alert
  • One-tap confirm (sets prep time) or reject (with reason)
  • Status board shows every stage in real time
  • Driver ETA shown once assigned
  • Pause restaurant for 15 / 30 / 60 min or indefinitely
🔒 Secure Handover
  • 6-character code generated per order
  • Driver shows code on their phone
  • Restaurant enters code to confirm pickup
  • Code mismatch blocks handover — audit trail created
  • Admin override available if code is lost
🍜 Menu & Onboarding
  • Import menu from any restaurant website URL
  • Photo of physical menu — AI reads it
  • Excel template — row-by-row upload
  • Toggle items available / unavailable on the fly
  • Stripe Identity + Connect set up in guided onboarding
