Admin App & Web Portal
Platform operations, oversight & partner management
Real Screenshots Live Platform Full Audit Trail 10 Background Jobs

The admin app gives operators complete visibility and control over the entire platform. Approve restaurants and drivers, override order states, resolve disputes, manage refunds, and monitor all activity — from a mobile app or web portal. Every admin action is logged to an immutable audit trail.

Admin Workflows
Onboarding Workflow
Application Received → Review Profile → Approve or Reject (with reason) → Partner Notified
Operations Workflow
Search User / Order → Inspect State → Override / Refund / Escalate → Audit Logged

Restaurant Management Dashboard. Full restaurant roster with status filters, revenue, and one-click approval / suspension controls. The web portal gives operators a larger viewport for bulk operations.
Immutable Audit Log. Every admin action — approval, rejection, order override, refund, manual state change — is logged with timestamp, actor, and payload. Read-only. Cannot be deleted.

Capabilities
✅ Onboarding & Trust

Approve or reject restaurant and driver applications with a written reason. Identity verification via Stripe. All decisions are logged to the immutable audit trail.

🔧 Order & Dispute Management

Override any of the 20+ order states. Initiate refunds directly from the admin panel. Escalate disputed orders. Full order history and item breakdown visible at a glance.

🔍 Monitoring & Search

Platform-wide search across users, restaurants, orders, and drivers. Background jobs continuously monitor for stuck states 24/7, alerting and auto-resolving without manual intervention.
