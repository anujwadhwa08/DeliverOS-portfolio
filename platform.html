MCP API — Agent-to-Agent Ordering (No App · No Account · Zero Friction)
New customer
Agent calls place_order with customerInfo + confirmDataSharing: true → DeliverOS creates profile, encrypts address, charges card → Returns live order ID + tracking
Existing customer
Agent calls place_order → OTP sent to customer's registered email → Customer shares code → agent calls verify_agent_identity → 15-min token returned → agent retries order, no re-verification needed
